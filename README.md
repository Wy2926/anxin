# 安心 · 守护家长 App

让子女远程守护、关怀并轻度监督父母日常健康与安全的家庭关怀 App。
核心理念：**关怀 > 监控**，所有敏感能力以父母知情、可控、可拒绝为前提。

## 可点击 Web 原型（P0）
位于本仓库根目录，使用 Vite + React + TypeScript，无后端，纯前端 mock，演示**子女端 ↔ 父母端联动**。

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # 生产构建
npm run lint     # 代码检查
```

演示要点：
- 「子女端」发起看屏 → 「父母端」弹出同意框（强同意）。
- 「父母端」用药提醒点「已经吃了」→「子女端」看板/消息实时打卡。
- 「父母端」SOS（3 秒倒计时可取消）→「子女端」收到紧急告警。
- 「父母端」点「我很好」报平安 →「子女端」更新平安时间。
- 「子女端」我的 →「编排父母端手机界面」：切模板、调字号/对比度/语音、排序与开关入口，含**实时预览**，点「推送到设备」→「父母端」首屏即时重排（M7 特色能力）。

## 核心能力（规划）
- 双端：子女端（Guardian）/ 父母端（Elder，极简适老）
- 实时语音 / 视频通话
- 远程协助看屏（强同意、限时、可审计）
- 用药提醒与服药确认闭环
- SOS 紧急求助、报平安、健康记录、定位/安全区
- 子女远程"编排"父母端首屏界面
