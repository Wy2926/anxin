# 安心 · 守护家长 App

让子女远程守护、关怀并轻度监督父母日常健康与安全的家庭关怀 App。
核心理念：**关怀 > 监控**，所有敏感能力以父母知情、可控、可拒绝为前提。

## 核心能力（规划）
- 双端：子女端（Guardian）/ 父母端（Elder，极简适老）
- 实时语音 / 视频通话
- 远程协助看屏（强同意、限时、可审计）
- 用药提醒与服药确认闭环
- SOS 紧急求助、报平安、健康记录、定位/安全区
- 子女远程"编排"父母端首屏界面
